# Mathermatical Operators ➕➖
There are different types of mathematical operator
* Addition(+)
* Subtraction(-)
* Multiplication(*)
* Division(/)
* Power(**)
* Divisor(//)

## Rules for Mathematical Operators
Python follows Pemdas rule
P- Parenthesis
E- Exponential
M- Multiplication
A- Addition
S- Subtraction
